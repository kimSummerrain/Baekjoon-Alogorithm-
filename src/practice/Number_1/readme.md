오버라이딩 vs 오버로딩
오버라이딩: 부모 메서드 재정의 (다형성 실현)
오버로딩: 같은 이름 메서드, 매개변수 다르게
추상 클래스 vs 인터페이스
추상 클래스: 공통 속성 + 일부 구현 가능
인터페이스: 규약 정의, 다중 구현 가능
